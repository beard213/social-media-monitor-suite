# Changelog

## 4.0.0 - 2026-07-15

- 项目更名为“社交媒体及网络直播监测与溯源系统”；
- 将页面调整为更亮的蓝色科技风，放大字号与关键指标，增加一键大屏模式；
- 将关键词短视频搜索与直播ID接入合并到“搜索与直播接入”页面；
- 新增平台、直播间ID、区域、关键词、分片时长和测试流地址配置；
- 新增直播服务桥接协议 `/v1/live/resolve`，可复用已经完成的直播监控服务；
- 无直播流时将会话保存为 `waiting_source`，获得流地址后可继续采集，不伪造成功；
- 新增公开关系扩列协议 `/v1/relations`，支持评论高频账号及经授权返回的关注、朋友、关联账号线索；
- 抖音和快手的搜索、评论、关系、媒体解析集中留给赵帅项目方填写；
- 增加向赵帅项目方推送事件JSON和媒体文件的环境变量与接口说明；
- 增加公开内容监测边界、人工复核和数据最小化说明；
- 自动化测试扩展到12项。

## 3.0.0 - 2026-07-15

- 重做监测大屏和九个业务页面；
- 新增评论弹幕监控、风险账号处置、AI研判、话题溯源；
- 新增 `/api/console/*` 控制台聚合接口；
- 新增可独立运行的抖音/快手授权连接器服务框架；
- 新增媒体地址刷新协议 `/v1/media/resolve`；
- 丰富虚构演示数据并在启动时自动初始化；
- 修复连接器日期入库、任务错误状态、ASR空文件和并发领取问题。
