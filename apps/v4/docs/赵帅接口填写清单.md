# 赵帅项目方接口填写清单

当前任务中心、前端大屏、数据库、任务队列、评论入库、扩列、检测、分片录制和推送框架已经完成。平台接口代码集中在以下两个文件：

```text
connector_service/drivers/douyin.py
connector_service/drivers/kuaishou.py
```

只需要实现四个方法，返回统一数据结构，主系统不再修改：

```python
search(request)         # 关键词批量搜索短视频/直播
comments(request)       # 获取某条视频的公开评论
relations(request)      # 获取博主公开关注/朋友/高频评论者等关系线索
resolve_media(request)  # 按视频ID/直播间ID返回新的媒体或直播流地址
```

## 1. 搜索接口

任务中心调用：

```http
POST /v1/search
Authorization: Bearer <CONNECTOR_INTERNAL_TOKEN>
Content-Type: application/json
```

请求：

```json
{
  "keywords": ["央企讨薪", "雄安"],
  "content_types": ["video", "live"],
  "regions": ["xiongan"],
  "time_range_hours": 24,
  "sort_by": "latest",
  "keyword_match_mode": "any",
  "limit": 50
}
```

返回：

```json
{
  "items": [
    {
      "platform_content_id": "平台视频ID或直播间ID",
      "content_type": "video",
      "title": "标题",
      "description": "描述",
      "source_url": "公开来源页",
      "media_url": "有权限时填写",
      "stream_url": "直播时可填写",
      "cover_url": "封面",
      "author_id": "平台公开账号ID",
      "published_at": "2026-07-15T10:00:00+08:00",
      "matched_keywords": ["央企讨薪"],
      "metadata": {
        "engagement": {"views": 1000, "likes": 20, "comments": 8, "shares": 2},
        "hashtags": ["讨薪"],
        "region_tags": ["雄安新区"]
      }
    }
  ],
  "cursor": null,
  "has_more": false
}
```

## 3. 公开关系扩列接口

```http
POST /v1/relations
```

请求：

```json
{"author_id": "平台公开账号ID", "limit": 100}
```

返回：

```json
{
  "relations": [
    {
      "account_id": "关联账号ID",
      "account_alias": "关联账号公开昵称",
      "relation_type": "friend",
      "profile_url": "公开主页",
      "evidence_count": 3,
      "source": "authorized_connector",
      "metadata": {}
    }
  ],
  "cursor": null,
  "has_more": false
}
```

`relation_type` 允许：

```text
following
follower
friend
frequent_commenter
related_account
```

系统只使用平台公开或正式授权返回的关系，不抓取私密关系，不推断现实身份或个人政治属性。

## 4. 媒体和直播流解析

```http
POST /v1/media/resolve
```

请求：

```json
{"platform_content_id": "直播间ID", "content_type": "live"}
```

返回：

```json
{
  "media_url": "",
  "stream_url": "https://...",
  "expires_at": "2026-07-15T11:00:00+08:00",
  "request_headers": {
    "User-Agent": "...",
    "Referer": "..."
  },
  "metadata": {}
}
```

## 5. 推送接收接口

主系统已有推送程序，只需要提供：

```env
PUSH_EVENTS_URL=http://赵帅服务/api/events
PUSH_MEDIA_URL=http://赵帅服务/api/media
PUSH_BEARER_TOKEN=双方约定的Token
```

事件接收接口应返回 HTTP 2xx；推荐返回：

```json
{"ok": true, "ack_id": "receiver-0001"}
```
