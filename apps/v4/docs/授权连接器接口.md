# 授权连接器统一接口 v1.0

任务中心不直接保存平台 `client_secret`、`access_token` 或签名逻辑。每个平台运行一个独立连接器服务，连接器负责官方凭证、Token刷新、分页、限流、字段映射和短时媒体地址刷新。

## 认证

任务中心请求连接器时携带：

```http
Authorization: Bearer <平台连接器内部令牌>
Content-Type: application/json
```

该内部令牌不是平台官方 Token。

## 1. 健康检查

```http
GET /health
```

建议响应：

```json
{
  "ok": true,
  "platform": "douyin",
  "credentials_configured": true,
  "capabilities": {
    "search": true,
    "comments": true,
    "live": false,
    "media_resolve": false
  }
}
```

## 2. 搜索

```http
POST /v1/search
```

请求：

```json
{
  "keywords": ["关键词A", "关键词B"],
  "content_types": ["video", "live"],
  "regions": [],
  "time_range_hours": 24,
  "sort_by": "latest",
  "keyword_match_mode": "any",
  "limit": 50
}
```

响应：

```json
{
  "items": [
    {
      "platform_content_id": "平台稳定内容ID",
      "content_type": "video",
      "title": "标题",
      "description": "描述",
      "source_url": "https://公开来源",
      "media_url": "",
      "stream_url": "",
      "cover_url": "https://封面",
      "author_id": "平台公开作者标识",
      "published_at": "2026-07-15T10:00:00+08:00",
      "matched_keywords": ["关键词A"],
      "metadata": {
        "engagement": {
          "views": 0,
          "likes": 0,
          "comments": 0,
          "shares": 0
        },
        "region_tags": [],
        "hashtags": []
      }
    }
  ],
  "cursor": null,
  "has_more": false
}
```

`published_at` 可以是 ISO 8601 字符串、Unix秒或 `null`，任务中心会统一转换为 `datetime`。

## 3. 评论

```http
POST /v1/comments
```

请求：

```json
{
  "platform_content_id": "平台稳定内容ID",
  "limit": 100
}
```

响应：

```json
{
  "comments": [
    {
      "platform_comment_id": "评论ID",
      "author_id": "公开作者标识",
      "text": "评论文本",
      "like_count": 0,
      "published_at": "2026-07-15T10:01:00+08:00",
      "metadata": {}
    }
  ],
  "cursor": null,
  "has_more": false
}
```

## 4. 媒体/直播流刷新

```http
POST /v1/media/resolve
```

请求：

```json
{
  "platform_content_id": "平台稳定内容ID",
  "content_type": "live"
}
```

响应：

```json
{
  "media_url": "",
  "stream_url": "https://短时有效授权地址",
  "expires_at": "2026-07-15T10:30:00+08:00",
  "request_headers": {
    "User-Agent": "连接器要求的请求头"
  },
  "metadata": {}
}
```

不要在搜索阶段长期缓存短时直播流地址。任务中心在采集前调用该接口刷新。

## 5. 直播互动主动推送

连接器把已获授权的公开直播互动事件发送到任务中心：

```http
POST /api/provider/ingest/live-message
X-API-Key: <ADMIN_API_KEY，若任务中心启用>
```

```json
{
  "platform": "douyin",
  "platform_content_id": "直播内容ID",
  "message_type": "comment",
  "author_id": "公开作者标识",
  "text": "弹幕文本",
  "event_time": "2026-07-15T10:02:00+08:00",
  "metadata": {
    "like_count": 0,
    "gift_name": "",
    "gift_count": 0
  }
}
```

`message_type` 可根据授权数据源使用：`comment`、`like`、`follow`、`gift`、`share` 等。

## 错误约定

- `401`：任务中心内部Token错误；
- `429`：上游限流；
- `501`：驱动尚未接入对应平台能力；
- `502/503`：上游授权平台暂时不可用；
- 错误响应使用 `{"detail":"说明"}`。
