# 前端流程优化版 V3 说明

## 一、本版目标

本版不重新更换领导已认可的浅色视觉风格，重点解决会议反馈中的信息架构与表达问题：

1. 左侧导航按照实际监测流程组织；
2. 删除“预警处置”一级页面；
3. 将“接口与系统”改造成“账号配置”和“运行记录”；
4. 搜索接入改成直观的任务类型入口与任务向导；
5. 实时弹幕与公开视频评论严格分开；
6. 智能分析采用“输入—过程—结果—依据”结构；
7. 知识图谱改成有来源依据的“线索关系图”；
8. 人工处理仅保留“待确认、已确认、已忽略”。

## 二、新左侧导航

```text
态势总览

监测准备
  账号配置
  监测任务

内容监测
  直播监测
  短视频监测

分析研判
  智能内容分析
  话题与线索溯源

运行管理
  运行记录
```

完整使用流程：

```text
账号接入
  ↓
创建监测任务
  ↓
查看直播或短视频结果
  ↓
查看输入、处理过程与检测结果
  ↓
形成有来源依据的话题和关联线索
  ↓
查看运行记录与错误原因
```

## 三、各页面改动

### 1. 态势总览

首页改为展示真实业务指标：

- 监测任务；
- 正在运行直播；
- 证据文件；
- 检测完成记录；
- 待确认内容；
- 失败任务；
- 24小时任务运行趋势；
- 最新监测内容；
- 热点话题；
- 最新运行记录；
- 平台与服务状态。

“检测完成”明确表示检测记录数，不等同于风险命中数。

### 2. 账号配置

原“系统管理”中的连接器和服务配置移入账号配置：

- 抖音账号能力；
- 快手账号能力；
- 直播采集服务；
- 内容检测服务；
- 结果同步服务；
- 测试连接；
- Token是否配置；
- 高级设置与接口契约。

新增、编辑账号和授权有效期暂时标记为待开发，不生成假账号配置。

### 3. 监测任务

新建任务时先选择类型：

```text
关键词监测
直播间监测
短视频链接
批量导入
```

已完成：

- 关键词任务向导；
- 快捷模板自动填充；
- 平台、内容类型和区域设置；
- 关注词与排除词；
- 评论采集、线索扩展、自动检测、媒体采集和结果同步选项；
- 直播链接自动提取房间ID；
- 直播间批量粘贴导入；
- `平台,房间ID/链接` 格式导入；
- 已有任务运行、暂停和删除。

短视频链接直连接口尚未实现，页面明确显示待开发，不使用关键词任务冒充。

### 4. 直播监测

直播监测明确分成六类内容：

```text
语音转写
视频片段
实时弹幕
公开视频评论
检测结果
运行日志
```

实时弹幕只显示 `kind=live_message` 的真实平台直播消息；公开视频评论单独显示，不再混入弹幕。

无真实弹幕时显示：

```text
实时弹幕接口尚未接入
当前已接入视频片段、音频提取和语音转写
真实弹幕需平台授权连接器接入
```

检测服务未启用时显示真实的“跳过”状态，并说明“跳过不代表安全或有害”。

### 5. 短视频监测

原“预警处置”中的内容列表迁入短视频监测：

- 短视频结果列表；
- 平台、关键词和人工状态筛选；
- 视频或证据播放；
- 内容描述和来源链接；
- 公开视频评论；
- 检测结果；
- 证据文件；
- 关联线索；
- 评论采集、重新检测和线索扩展；
- 人工状态：待确认、已确认、已忽略。

页面不再使用“处置”“封禁”或“敌对账号”等表述。

### 6. 智能内容分析

取消聊天式AI表达，采用四个固定区域：

```text
1. 输入内容
2. 分析流程
3. 分析结果
4. 判断依据
```

分析结果明确展示：

- 分析类型；
- 检测标签；
- 模型置信度；
- 人工确认状态；
- 模型名称；
- 模型版本；
- 分析时间；
- 证据文件；
- 转写原文和命中关键词。

### 7. 话题与线索溯源

不再展示没有证据支持的复杂知识图谱，只展示：

- 话题趋势；
- 关键词命中关系；
- 内容共现关系；
- 关联内容；
- 首次发现和最近发现时间；
- 评论数量和线索数量；
- 关系来源与内容描述。

评论共现和外部公开关系没有真实接口时，页面明确提示尚未接入。

### 8. 运行记录

原“系统管理”中的任务队列迁入运行记录：

- 搜索任务；
- 评论采集；
- 公开关系采集；
- 线索扩展；
- 文本检测；
- 媒体采集；
- 直播分片；
- 结果同步；
- 平台与服务状态；
- 错误原因。

任务参数和原始错误默认折叠，避免普通页面直接展示大量JSON。

## 五、仍需后端新增开发

1. 平台账号持久化、新增、编辑、停用；
2. 授权有效期和Token刷新；
3. 单条短视频链接直连接入接口；
4. 真实抖音/快手直播弹幕接口；
5. 直播分片统一 `segment_id` 和句级ASR时间轴；
6. 评论账号共现与公开关系来源详情；
7. 正式报告保存、版本和审核记录；
8. 用户角色、后端登录和操作审计。
